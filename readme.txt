【基本コマンド】

・Fractalをインストール
　npm install --save @frctl/fractal

・FractalのCLIツールをインストール
　npm i -g @frctl/fractal

・localhost:3000 に表示(自動コンパイル)
　fractal start --sync

・静的ファイルの書き出し （fractal.jsの設定が必要）
　fractal build
